## 🏠 홈 페이지 구성

### 1. 헤더
- **인사말**: "안녕하세요, 홍길동님 👋"
- **프로필**: 우측 상단 원형 아바타
- **소속**: "고3 N반 · ACE ENGLISH 강남점"

### 2. 오늘의 모의고사
- **상태 배지**: "진행중" (Indigo)
- **제목**: "2024년 9월 모의고사 변형"
- **정보**: 시간, 문제 수
- **CTA**: "바로 응시하기" 버튼

### 3. 마감 임박 숙제 (2개)
- **카드 형태**: 작은 카드 (2개 리스트)
- **상태**: "오늘 마감" / "내일 마감" (Amber)
- **정보**: 제목, 마감 시간, 문제 수
- **인터랙션**: 클릭 가능 (ChevronRight 아이콘)

### 4. 최근 성적
- **2개 카드**: 최근 시험 점수, 평균 점수
- **추세 표시**: TrendingUp 아이콘 + "+3.2"

### 5. 오늘의 복습
- **오답 10문제**: 큰 카드 + "시작" 버튼
- **약한 영역 TOP3**: Amber 배지 (순위 표시)
